# homing-pigeon

Quantum Theory applied to Chemistry

Read literature and write code for Quantum Theory applied in Chemistry

## Goals

1. complete [Quantum Theory for Chemical Applications by Jochen Autschbach][qtca]
2. complete [Molecular Electronic Structure Theory by Trygve Helgaker][mest]

## Tools

- [Hasktorch][ht] functional differentiable programming in Haskell

## Why homing-pigeon?

Professor Autschbach gave me [sct][Schrodinger's Cat Trilogy] for completing my PhD studies.
The final book is called "The Homing Pigeons".
